Utilisation d'un Spinner

Les curseurs (jSlider) et les spinners font tous deux la même chose. Ils permettent à l'utilisateur de choisir une valeur dans un intervalle prédefini.

Posez un spinner sur votre feuille.

Dans les propriétés repérez celle qui se nomme model, cliquez sur le bouton de suite (... " points) à droite de la propriété, une fenêtre s'ouvre


Vous pouvez choisir plusieurs modèles (entre autre un modèle qui permet de choisir les dates, ce qui est toujours pénible à implanter en informatique), mais ici nous choisissons Number



Est-il nécessaire d'en dire plus, vous choisissez, la valeur initiale, le Min et le Max ainsi que le pas :



Utilisation de la valeur sélectionnée :

On obtient la valeur sélectionnée avec la méthode :

jSpinnerGain.getValue()

Mais cette méthode retourne une valeur de type "object" car ce que retoune un spinner peut aussi bien être un nombre qu'une date etc...

Il faut donc convertir cet objet en chaîne de caractères :

String.valueOf(jSpinnerGain.getValue())

Puis convertir cette chaîne en entier pour récupérer la valeur :

Integer.parseInt(String.valueOf(jSpinnerGain.getValue()))

Intuitif, non ?
